## Affiliate Dashboard (`/affiliate/dashboard`)

### Overview Tab (Default)
The overview is a customizable widget grid. Click "Customize" to add/remove widgets.

| Widget | What It Shows |
|--------|--------------|
| Referral Link | Your unique referral URL with copy button |
| Live Earnings | Real-time Today/Week/Month earnings |
| Quick Stats | Clicks, Signups, Pending, Total Earned |
| Tier Progress | Progress bar toward next tier (Bronze → Silver → Gold → Platinum) |
| Milestone Bonuses | Progress toward referral milestones with cash rewards |
| Active Contests | Current competitions with prizes and countdowns |
| Earnings Forecast | Projected monthly earnings with optimistic/pessimistic range |
| Leaderboard | Your rank vs other affiliates |
| Conversion Funnel | Visual breakdown: Clicks → Signups → Trials → Paid |
| AI Coach | Personalized performance tips based on your data |
| Content Analytics | YouTube/social integration stats (requires connected platforms) |
| Achievements | Grid of earned badges |
| Milestone Countdown | Specific tracker for next bonus threshold |
| Payout Schedule | Next payment date and eligibility status |
| Tier Celebration | Special notice when you reach a new tier |

### Analytics Tab
Scroll through these sections (in order):
1. **Earnings Over Time** — Line chart of earnings history
2. **Conversion Funnel** — Detailed bar chart of each conversion stage
3. **Earnings Heatmap** — GitHub-style 52-week activity grid
4. **Performance Benchmarks** — Gauge showing your percentile vs program average
5. **Expanded Analytics** — Click heatmap by day/hour, conversion by channel, month-over-month scorecard, personal best, efficiency metrics, dual-axis chart, campaign comparison
6. **Flywheel Analytics** — Churn rate, churn reasons, churn timing, at-risk alerts, net growth, retention curve, conversion trends, trial benchmarks, revenue pie chart, cumulative earnings, AI insights panel
7. **Flywheel Reports** — Connected analytics dashboard, financial overview, predictive intelligence, content intelligence, custom date range reports

### Referrals Tab
- **List View** — All referred users with status (active, trial, churned)
- **Portfolio View** — Toggle to see portfolio value, avg LTV, active subscribers
- Sparklines in each row show recent performance direction

### Earnings Tab
- **Status Cards** — Pending, Approved, Paid Out amounts + minimum payout threshold
- **Commission List** — Expandable items showing the 7-step commission lifecycle (Click → Signup → Trial → Paid → Commission → Approved → Paid Out)

### Payouts Tab
- **Payout History** — Filterable, paginated list with CSV export
- **Earnings Goal Setter** — Set monthly/quarterly targets with progress tracking
- **Referral of the Month** — Spotlight on featured partner
- **Affiliate Manager Card** — Contact info for support
- **Grace Period Notice** — Churn grace period policy
- **Weekly Challenges** — Active challenges with progress bars and badge rewards

### Resources (Assets) Tab
- **Search and Filter** — Find assets by title or type
- **Asset Cards** — 8 types: banners, social posts, email templates, documents, videos, bundles
- **Top Performer Badges** — Visual badges on highest-converting assets
- **Asset Usage Tracking** — Download/copy/view counts
- **Swipe File Library** — Pre-written email copy with merge tags ({affiliate_name}, {referral_link}, {discount_code})
- **Case Study Library** — Success stories with key metrics, quotes, and share buttons

### Tools Tab
- **Deep Link Generator** — Create tracked links to any site page with UTM parameters
- **QR Code Generator** — Download branded QR codes for your referral link
- **Link Presets** — Save/load frequent link configurations
- **Link Shortener** — Create clean short URLs (e.g., `/r/my-link`)
- **Promo Codes** — View and manage your personalized discount codes
- **Media Kit** — Professional partner page with your stats
- **Copy-Paste Captions** — Pre-written social posts for 5 platforms with your code auto-inserted
- **Starter Kit** — Essential materials bundle for getting started
- **Commission Disputes** — File and track missing/incorrect commission disputes
- **Promotion Quiz** — AI-powered 6-question quiz that generates a personalized 30-day playbook

### News (Announcements) Tab
- System announcements and program updates from administrators

### Messages Tab
- Direct messaging with program administrators
- Unread message indicators
- **Affiliate Directory Preview** — Browse other affiliates in the program

### Account Tab
- **Profile Settings** — Name, avatar, bio, payout method
- **Tax Center** — Annual earnings summary, estimated withholding, 1099 status, W-9/W-8BEN submission
- **Promotional Calendar** — Upcoming campaigns with countdowns and content suggestions

### Support Tab
- **Knowledge Base** — Searchable help articles organized by category (10 articles covering getting started, tools, and tips)

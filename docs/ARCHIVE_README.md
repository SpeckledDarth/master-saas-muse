# PassivePost Documentation

PassivePost is an AI-powered social media management platform that helps creators and businesses automate content creation, scheduling, and cross-platform publishing. Built on the MuseKit SaaS template, it provides a complete flywheel system for turning blog posts into social media content with engagement tracking, lead generation, and revenue attribution. The platform includes a full affiliate program, subscription billing, and admin dashboard out of the box.
